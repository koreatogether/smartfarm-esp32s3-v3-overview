HTTP API

AP 접속 후 사용 가능한 엔드포인트와 인증 방식입니다.

GET /api/ping

서버 생존 확인

GET /api/state

현재 보드 전체 상태 조회

POST /api/command

제어 명령 실행 (X‑API‑Token 필요)

State JSON Example 
{
  "ok": true,
  "board": "SmartFarm Control Node 03",
  "wifi": {
    "mode": "AP",
    "connected": true,
    "ip": "여기에 기입",
    "ssid": "여기에 기입"
  },
  "state": {
    "motor": "Stop",
    "lcdBacklight": false,
    "lcdCountdown": 0,
    "ledColor": "green",
    "buttonState": "HIGH",
    "buttonLatched": false
  }
}

Command Reference

POST /api/command 에 전달하는 target / value 조합입니다.

Target Value Action
motorforward모터 전진
motorstop모터 정지
ledgreenLED 초록색
ledblueLED 파란색
ledyellowLED 노란색
ledredLED 빨간색
ledwhiteLED 흰색
lcdonLCD 백라이트 ON (60s 카운트다운)
lcdoffLCD 백라이트 OFF
buttonsimulate버튼 입력 시뮬레이션
curl Example 
curl -X POST http://여기에_기입/api/command \
  -H "Content-Type: application/json" \
  -H "X-API-Token: 여기에_기입" \
  -d '{"target":"motor","value":"forward"}'
